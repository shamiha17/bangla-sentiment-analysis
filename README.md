# Bangla Sentiment Analysis

This project implements **sentiment analysis for Bangla text** using machine learning.  
It predicts whether a given Bangla text has a **positive**, **negative**, or **neutral** sentiment.


## Features

- Handles Bangla text input
- Uses TF-IDF vectorization for feature extraction
- Supports multiple ML models: Logistic Regression, Random Forest, etc.
- Generates predictions in CSV format
